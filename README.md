# Project Config

Configuration generator for multi-agent dev projects. Fill out a form → export config → send to Hermes → scaffold is auto-populated.

## Stack
- Next.js 15 (static export) · TypeScript · Tailwind v4 · pnpm

## Quick Start
```bash
pnpm install && pnpm dev
```

See [AGENTS.md](AGENTS.md) for the multi-agent workflow.
